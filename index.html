My projects

SeGuard

SeGuard is a project that classifies Android applications. My role in this project is to take the semantic graphs from my supervisor, and parse(featurize) it into vectors. These vectors contains information (topological features) of the graph that we think is crucial for the classicication. Then, we feed it to the downstream random forest classifier. Up to now, we have about 70% of accuracy for binary classification (Answer the question: is it a benign or malicious application).

Classical Music Genre Classification

Our team is working on a classical music genre classifier. The classifier should be able to distinguish music from different time periods. e.g. romantic or baroque. Our approach is to use 3-second spectrogram as a feature, and process it using LSTM layer. We hope our neural network is able to connect the "dots" across timestamp and decide which style this particular 3 second belongs to.
